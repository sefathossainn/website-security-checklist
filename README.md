# Website Security Checklist
A complete checklist to help you secure your WordPress or any other CMS-based website.  
This guide covers the essential steps for preventing hacking, malware infection, and spam attacks.
## Basic Security Measures
 - Always keep WordPress core, themes, and plugins updated  
 - Use strong admin usernames and passwords  
 - Disable file editing from the WordPress dashboard  
 - Change your database prefix from `wp_` to something unique  
 - Set correct file and folder permissions (644 / 755)
# Advanced Protection
-  Install a security plugin such as **Wordfence**, **Sucuri**, or **iThemes Security**  
-  Regularly scan your website for malware  
-  Use **Cloudflare** or similar firewall (WAF) to block suspicious IPs  
-  Disable XML-RPC to prevent brute-force attacks  
-  Change all passwords after any hack attempt  
-  Enable **two-factor authentication (2FA)** for all admin accounts  
# Backup & Recovery
-  Use plugins like **UpdraftPlus** or **JetBackup** for automated backups  
-  Store backups offsite (Google Drive, Dropbox, or your hosting backup system)  
-  Clean old backups regularly to avoid clutter
